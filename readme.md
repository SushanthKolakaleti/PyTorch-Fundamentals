# PyTorch Fundamentals

This repository contains various PyTorch experiments for learning the fundamentals of neural networks using the PyTorch library of python.

## Requirements

Install the required dependencies with:
    pip install -r requirements.txt


## How to Run

Open the notebook:
    jupyter notebook "filename/notebooks/file.ipynb"
Then run the cells from top to bottom


## Note :

Every notebook in this repo runs on the Kaggle environment. For local running, make the required changes for clean running of the code
